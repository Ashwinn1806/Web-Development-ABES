Understanding Basic CSS Properties

In this example, we will learn how to use CSS to style:

  • Text
  • Colors
  • Fonts
  • Margin
  • Border
  • Padding
  • Width and Height

What Each Property Does:

Text: Changes how text appears â€” for example, bold, italic, underline, or text alignment.

Color: Defines text color or background color.

Font: Sets the typeface and size of the text.

Margin: Creates space outside the border (like outer spacing).

Padding: Creates space inside the border (between text and border).

Border: Adds a visible outline around an element.

Width and Height: Control the size of elements.
