Understanding HTML Form Attributes

HTML forms use several attributes to control how data is collected and submitted. Below are the most common and useful form attributes explained with examples.

1. method Attribute

The method attribute specifies how form data is sent to the server. It can have two values:

  • GET — Appends form data to the URL (used for simple or read-only requests).
  • POST — Sends form data securely in the request body (used for sensitive data).

2. action Attribute

The action attribute defines the destination (URL or page) where the form data will be sent after submission.
For example: action="submit_page.html"

3. placeholder Attribute

The placeholder attribute displays a hint inside input fields to guide users about what to enter.

4. required Attribute

The required attribute makes a field mandatory — the user cannot submit the form without filling it.
