Understanding HTML Tables

HTML tables are used to organize data in rows and columns. A table is created using the <table> tag, and rows are added using <tr> (table row). Inside each row, cells are defined using <td> (table data) or <th> (table header).

Important Table Attributes:

  • border: Defines the thickness of table borders.
  • cellspacing: Space between table cells.
  • cellpadding: Space inside each cell (between content and border).
  • caption: Adds a title to the table.
  • rowspan: Merges cells vertically (across rows).
  • colspan: Merges cells horizontally (across columns).

Note: Tables are very useful for creating structures like timetables, results, or data reports.
